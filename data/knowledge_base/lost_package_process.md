# Lost package process

Policy code: LOG-011

A package is classified as "lost" (not delayed) when ANY of the following
apply:

- Tracking shows "delivered" but customer confirms non-receipt
- Tracking has shown no movement for 10+ consecutive days
- Courier confirms the package cannot be located

Process:
1. File a lost package claim with the courier via the internal logistics
   portal within 24 hours of confirming lost status
2. Offer the customer immediate reshipment at no cost, do not wait for the
   courier claim to resolve
3. If the item is out of stock for reshipment, offer a full refund instead,
   regardless of original purchase date or tier — lost package always
   overrides the standard 14-day or premium 30-day refund windows
4. Lost package cases do not require supervisor escalation unless the
   customer has had a prior lost package claim in the last 90 days, in
   which case escalate under ESC-002

Do not confuse this with the delivery delay process (DEL-007) — a delayed
package is still in transit, a lost package is not trackable or was marked
delivered without confirmation.
