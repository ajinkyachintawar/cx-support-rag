# Return shipping cost responsibility

Policy code: RET-SHIP-018

For approved refunds, who pays return shipping depends on the reason:

- Service failure (late delivery, billing error, wrong item sent, damaged
  on arrival): we provide a free prepaid return label, customer pays
  nothing
- Customer changed their mind, ordered wrong size, or no longer wants the
  item (no service failure): customer pays return shipping, deducted from
  the refund amount or paid upfront, agent's discretion based on item value
- Premium tier customers always get free return shipping regardless of
  reason, as a tier benefit, even for change-of-mind returns

The prepaid label is generated through the internal shipping tool using
code RET-SHIP-018 as the reference reason code, this links the return to
the correct cost center for reporting.
