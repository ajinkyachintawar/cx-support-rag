# Account verification procedure

Policy code: VER-003

Before discussing any order, refund, or account detail, agents must verify
the customer's identity using TWO of the following three:

- Full name matching account record
- Email address on file
- Order number or order confirmation code (format ORD-XXXXXX)

Phone number alone is NOT sufficient verification, even if it matches the
account, since phone numbers can be spoofed or shared within households.

If the customer cannot provide two matching identifiers, do not proceed
with account-specific discussion. Offer to send a one-time verification
code to the email on file instead. This is the only acceptable fallback
verification method.

Verification code format: 6 digits, expires in 10 minutes, sent via the
internal notification system, not visible to the agent.
